# 笔记声明

    不必特别在意规律顺序，甚至你也不用在意你学到了什么东西，你只需要按照这上面的一些内容来进行操作，只要你认为有兴趣，感觉这个东西很新奇，你就已经上到了，你不必很在意是否是很具体化的系统化的东西，因为那个东西不适合新手期的你来进行学习

# 认识你的系统

neofetch

这个指令可以帮你看到系统，版本，分辨率，内存，cpu，gpu等你可以知道的一些信息，甚至可以看到一些小的彩蛋，比如你在Linux Mint版本中输入，你还可以看到logo

# 内核版本

```
uname -a
```

你可以看到内核版本是什么样子的，比如版本号，编译序号，系统架构，操作系统等


# 查看版本

```
lsb release -a 
```

基于 Debian/Ubuntu 的发行版是可以直接使用的，但是其他版本需要使用他们的，或者另行安装即可。

# 关于文件的操作

```
ls
```

看看你这个文件夹里面有什么东西

```
ls -l
```

具体查看一下

```
cd 文件夹名
```

进入到（文件夹名）中

```
cd ..
```

返回到上一级文件夹中

```
cd ~
```

```
touch 文件名.xxx
```

比如txt，md等后缀都是可以的

```
mkdir 文件夹名
```

这个指令是专门创建文件夹的

```
cp 源文件 目标位置
```

这个指令是你用来复制在目标位置的源文件的

```
mv 源文件 目标位置
```

将源文件移动到目标位置

```
rm 文件名
```

```
zip -r 我的压缩包.zip 我的文件夹
```
进行文件的压缩

```
unzip 我的压缩包.zip
```

解压压缩包

# 文件的属性操作

```
cat 文件名
```

你就可以查看这个文件的一些信息了

```
less 文件名
```
这个指令可以帮你智能看文件，按“↑↓”翻页，习惯给好一点，按“q”退出

```
nano 文件名
```

```
find ~ -name “文件名.后缀”
```
可以从家目录里查找文件，查找到以后会给你显示出路径来



你就可以查看内容了,并且你在查看内容的同时，也是可以对其进行最基本的编辑，和git使用的vim不一样，用的是“nano”，这点要注意一下，不过无伤大雅，底下也会提醒你，比如你要退出，就会有“^X”,也就是ctrl+x就可以了，最后输入y（yes），按下enter就可以退出了，就这么简单，其他的也可以继续探索一下

# 装软件

```
sudo apt update
```

用到这个命令之后你就有了最高权限了，可以开始安装软件了，这个还是每个系的版本是不一样的，我使用的Linux Mint是Ubuntu系，所以可以使用这条指令

```
sudo apt install 软件名
```

利用这个来进行软件的安装，建议先问问ai有什么东西可以安的，再去安,这里我推荐两个，一个是“vlc”是万能播放器，另一个是“gimp”，这个是ps轻量版，可以玩一玩

# 办公

```
libreoffice
```

这个是Linux Mint自带的办公软件，其他版本的话去问ai，可以在终端打出来，也可以去搜索栏给搜索出来手动打开

# 随便玩玩的小命令

```
free -h
```

看看内存用了多少

```
df -h
```

看看硬盘用了多少

```
top
```

看看谁在占用cpu，有点像是任务管理器，别忘了按“q”退出

```
history
```

看看曾经敲过的命令

```
reboot
```

这条命令用来重启

```
shutdown now
```

立刻关机

```
uptime
```

查看开机了多长时间

```
lscpu
```

查看cpu的具体情况，型号等

```
alias ll ='ls -l'
```

可以自己自定义代码，把后面的定义到前面的去，这样的话，只敲前面的就可以出来原本的代码效果，也是属于个人定制的一部分哦

```
clear
```

如果终端看起来特别乱，直接clear清除掉就好了

# 好玩的小彩蛋

```
sudo apt install sl cowsay
```

先装上这个东西，然后可以分别输入以下两条指令

sl

# 权限

```
ls -l 文件名.后缀
```

可以通过这条指令运行以后查看文件的权限

```
chmod +x 文件名.后缀
```

可以通过这样的方式来给文件加入可运行的权限，并且是开放给所有用户的
